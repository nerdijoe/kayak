## How to deploy

### Server Express.js
```
// git clone the project
$ cd server
$ npm install

// migrate MySQL database
$ sequelize db:migrate

$ npm start

```

### Kayak Client React.js
```
$ cd kayak-client
$ npm install
$ npm start
```

### Admin Client React.js
**Use admin-client-v2, this admin client is running on localhost:3009**
```
$ cd admin-client-v2
$ npm install
$ npm start
```
